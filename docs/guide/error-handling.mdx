## Exception Classes

```python
from remem import (
    RememError,             # base ŌĆö catches everything
    AuthenticationError,   # 401 ŌĆö bad API key
    PlanLimitError,        # 402 ŌĆö hit memory limit
    MemoryNotFoundError,   # 404 ŌĆö memory ID doesn't exist
    DuplicateMemoryError,  # 409 ŌĆö duplicate content detected
)
```

---

## Handling Each Error

```python
from remem import RememClient, AuthenticationError, PlanLimitError, DuplicateMemoryError, RememError

client = RememClient(api_key="rm_live_xxx")

try:
    result = client.remember(
        "User prefers dark mode",
        user_id="user_123",
        agent_id="support_bot",
    )

except AuthenticationError:
    # Invalid or revoked API key
    print("Check your API key at remem.online")

except PlanLimitError:
    # Hit the memory limit for your plan
    print("Memory limit reached ŌĆö upgrade at remem.online/pricing")

except DuplicateMemoryError:
    # Content already exists ŌĆö this is fine to ignore
    pass

except RememError as e:
    # Catch-all for any other API error
    print(f"API error {e.status_code}: {e.detail}")
```

---

## Error Reference

| Exception | HTTP Code | When It Happens | What To Do |
|-----------|-----------|-----------------|------------|
| `AuthenticationError` | 401 | Invalid or missing API key | Check your key |
| `PlanLimitError` | 402 | Hit memory limit for your plan | Upgrade plan |
| `MemoryNotFoundError` | 404 | Memory ID doesn't exist | Check the ID |
| `DuplicateMemoryError` | 409 | Near-identical memory exists | Skip or ignore |
| `RememError` | 5xx | Server error | Retry the request |

## Retry on Server Errors

```python
import time
from remem import RememClient, RememError

client = RememClient(api_key="rm_live_xxx")


def remember_with_retry(content, user_id, agent_id, retries=3):
    for attempt in range(1, retries + 1):
        try:
            return client.remember(content, user_id=user_id, agent_id=agent_id)
        except RememError as e:
            if e.status_code >= 500 and attempt < retries:
                time.sleep(2 ** attempt)  # 2s, 4s backoff
                continue
            raise
```
