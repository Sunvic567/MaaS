## Step 3 — Store Your First Memory

```python
from memlayer import MemLayerClient

client = MemLayerClient(
    api_key="ml_live_xxx",
    base_url="https://memlayer.online",
)

result = client.remember(
    "User prefers concise bullet points over long paragraphs",
    user_id="user_123",
    agent_id="support_bot",
    memory_type="semantic",
    importance=0.8,
)

print(result.id)  # 775263ee-73af-4416-9804-1f274048ae08
```

<Check>
  You just stored your first memory. MemLayer embedded the text and saved it to your account.
</Check>

---

## Step 4 — Retrieve It

```python
# Load context at the start of a session
context = client.context(
    user_id="user_123",
    agent_id="support_bot",
)

for m in context.memories:
    print(m.content, m.importance)
```

---

## Step 5 — Search Semantically

```python
# Search with a natural language query
memories = client.recall(
    "what are this user's communication preferences?",
    user_id="user_123",
    agent_id="support_bot",
)

for m in memories:
    print(f"[{m.score:.3f}] {m.content}")

# Output:
# [0.891] User prefers concise bullet points over long paragraphs
```

<Check>
  The search found the memory even though the query used different words than the stored content. That's semantic search working.
</Check>
